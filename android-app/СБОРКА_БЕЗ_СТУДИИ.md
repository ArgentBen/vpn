# Сборка APK без Android Studio

## Способ 1: Через GitHub (рекомендуется)

Собрать APK можно бесплатно на GitHub, без установки Android Studio.

### Шаг 1: Залить проект на GitHub

1. Зарегистрируйтесь на [github.com](https://github.com), если ещё нет аккаунта.
2. Создайте новый репозиторий (например, `vpn-config`). Репозиторий может быть приватным.
3. Залейте туда папку проекта (всю папку `ВПН` или только `android-app` + `.github`).

   **Важно:** файлы `libv2ray.aar`, `geoip.dat`, `geosite.dat` большие (десятки МБ). GitHub может ругаться на файлы > 100 МБ. Варианты:
   - Залить всё (если репозиторий не отказывается),
   - либо залить только код **без** `app/libs/` и `app/src/main/assets/` — тогда в workflow нужно будет добавить шаги скачивания этих файлов (см. ниже).

   Если заливаете **всё** (включая libs и assets):
   - Откройте репозиторий на GitHub.
   - Перейдите во вкладку **Actions**.
   - Слева выберите workflow **"Build Android APK"**.
   - Нажмите **"Run workflow"** → **"Run workflow"**.
   - Дождитесь окончания (жёлтый кружок сменится на зелёную галочку).
   - Откройте последний запуск → внизу блок **"Artifacts"** → скачайте **app-debug-apk** (это ZIP с APK внутри; внутри будет `app-debug.apk`).

### Если не заливали libs и assets

Тогда в репозитории не должно быть папок с AAR и датами. В workflow нужно один раз добавить шаги, которые перед сборкой скачивают:
- `libv2ray.aar` из [релизов AndroidLibXrayLite](https://github.com/2dust/AndroidLibXrayLite/releases),
- `geoip.dat` и `geosite.dat` из релизов v2fly/v2ray-core (например, из `v2ray-extra.zip`),

и кладут их в `android-app/app/libs/` и `android-app/app/src/main/assets/`. Тогда сборка на GitHub будет работать без этих файлов в репозитории.

## Способ 2: На своём ПК (только командная строка)

Без Android Studio, но с установленными JDK и Android SDK.

1. Установите **JDK 17** (например, [Adoptium](https://adoptium.net/)).
2. Установите **Android command line tools** (без полной студии):  
   [developer.android.com/studio#command-line-tools-only](https://developer.android.com/studio#command-line-tools-only)
3. Установите **Gradle** 8.x с [gradle.org](https://gradle.org/releases/) или через пакетный менеджер.
4. В командной строке:
   ```bash
   cd "d:\Заказы\ВПН\android-app"
   gradle wrapper
   gradlew.bat assembleDebug
   ```
5. APK будет в `app\build\outputs\apk\debug\app-debug.apk`.

Переменная **ANDROID_HOME** должна указывать на каталог с Android SDK (где лежат `platforms`, `build-tools` и т.д.). Без этого сборка не пройдёт.
