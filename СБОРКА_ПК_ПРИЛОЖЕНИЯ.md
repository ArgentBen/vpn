# Сборка приложения для Windows (одна папка, exe)

Получается **одна папка** `dist\VPN_Obhod_app` с исполняемым файлом **VPN_Obhod_app.exe** — её можно копировать на другой ПК или раздавать. Python на целевом ПК не нужен.

## Что нужно на ПК для сборки

- Установленный **Python 3.10+** и **pip**
- Папка **v2ray** с файлами **v2ray.exe**, при необходимости **geoip.dat**, **geosite.dat** (у вас уже есть)

## Как собрать

1. Откройте папку проекта в командной строке (или дважды щёлкните по `build_windows.bat`).
2. Запустите:
   ```bat
   build_windows.bat
   ```
   Или вручную:
   ```bat
   pip install -r requirements.txt
   pip install pyinstaller
   python -m PyInstaller --noconfirm --clean vpn_app.spec
   xcopy /E /I /Y v2ray dist\VPN_Obhod_app\v2ray
   copy /Y ПРИЛОЖЕНИЕ_ПК_README.txt dist\VPN_Obhod_app\README.txt
   ```
3. Готовое приложение будет в папке **`dist\VPN_Obhod_app`** (файл **VPN_Obhod_app.exe**). Ничего закрывать не нужно — сборка идёт в новую папку.

## Что в итоговой папке `dist\VPN_Obhod`

- **VPN_Obhod_app.exe** — запуск приложения
- **v2ray/** — ядро V2Ray (v2ray.exe и др.), обязательно оставлять рядом с exe
- Остальные файлы и папки — зависимости (удалять не нужно)

Папку **`VPN_Obhod_app`** целиком можно копировать на другой компьютер с Windows и запускать **VPN_Obhod_app.exe** без установки Python.
