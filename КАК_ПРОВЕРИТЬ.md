# Как проверить, что V2Ray и прокси работают

---

## Весь ПК через V2Ray (рекомендуется)

Чтобы **все программы** (браузер, мессенджеры, игры и т.д.) ходили в интернет через V2Ray:

1. **Запустите V2Ray** — дважды кликните **`run_v2ray.bat`** и не закрывайте окно.
2. **Включите системный прокси** — дважды кликните **`Включить_прокси_весь_ПК.bat`**.
3. Готово. Весь трафик Windows пойдёт через 127.0.0.1:3128 → V2Ray → ваш прокси (8.221.141.88:50).

**Отключить:** запустите **`Выключить_прокси_весь_ПК.bat`** — прокси для системы выключится.

Часть программ подхватит прокси сразу, часть — после перезапуска. Если что-то не идёт через прокси, перезапустите эту программу.

---

## Только браузер (ручная настройка прокси)

## 1. Сначала включите прокси в браузере

Пока браузер не настроен на прокси, он не использует V2Ray — трафик идёт мимо. Нужно указать: **SOCKS5**, **127.0.0.1**, порт **1081**.

### Chrome или Edge

1. Нажмите **три точки** (меню) → **Настройки**.
2. В поиске настроек введите **прокси**.
3. Откройте **«Настройка прокси-сервера»** (или «Открыть настройки прокси-сервера компьютера»).
4. Откроется окно Windows «Параметры» → **Сеть и Интернет** → **Прокси**.
5. Включите **«Использовать прокси-сервер»**.
6. Укажите:
   - **Адрес:** `127.0.0.1`
   - **Порт:** `1081`
7. Сохраните (Chrome/Edge используют системный прокси).

**Важно:** в Windows по умолчанию задаётся HTTP-прокси, а V2Ray отдаёт SOCKS5. Поэтому лучше настроить прокси **внутри браузера** (расширением), а не только в Windows.

### Проще всего — расширение для прокси (рекомендуется)

1. Установите расширение, например **SwitchyOmega** (Chrome/Edge) или **FoxyProxy** (Chrome/Firefox).
2. Создайте профиль прокси:
   - **Тип:** SOCKS5
   - **Сервер:** 127.0.0.1
   - **Порт:** 1081
3. Включите этот профиль (переключатель в расширении).
4. Обновите страницу (F5) и откройте YouTube или любой сайт.

### Firefox

1. **Настройки** (три полоски) → **Основные** → внизу **«Параметры сети»** → **Настроить…**.
2. Выберите **«Ручная настройка прокси»**.
3. **SOCKS-хост:** `127.0.0.1`, **Порт:** `1081`.
4. Выберите **SOCKS v5**.
5. Оставьте «Использовать этот прокси для всех протоколов» или отметьте по необходимости.
6. **OK** → **OK**.

## 2. Проверка, что трафик идёт через прокси

1. **V2Ray должен быть запущен** (`run_v2ray.bat` — окно не закрывать).
2. В браузере **включён** прокси 127.0.0.1:1081 (SOCKS5).
3. Откройте в браузере сайт проверки IP, например:
   - https://api.ipify.org — покажет только ваш IP одной строкой
   - или https://2ip.ru
4. Если прокси работает, вы увидите **не ваш домашний IP** (часто это IP сервера 8.221.141.88 или его провайдера). Если IP не изменился — прокси в браузере не включён или не тот порт.

---

## 3. Если YouTube всё равно не открывается

- Убедитесь, что в браузере **действительно выбран** профиль с 127.0.0.1:1081 и он **включён**.
- Проверьте, что окно V2Ray **не закрыто** и в нём нет красных ошибок.
- Сервер 8.221.141.88:50 мог отключиться или ограничить доступ — тогда нужно уточнять у того, кто выдал вам эту ссылку.

## Кратко

| Шаг | Действие |
|-----|----------|
| 1 | Запустить `run_v2ray.bat` и не закрывать окно |
| 2 | В браузере указать прокси **SOCKS5**, **127.0.0.1**, **1081** (через расширение или настройки) |
| 3 | Открыть https://api.ipify.org — если IP изменился, трафик идёт через прокси |
| 4 | Открыть YouTube — запросы пойдут через 127.0.0.1:1081 → V2Ray → 8.221.141.88:50 |
