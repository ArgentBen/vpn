VPN Обход блокировок — приложение для Windows
==============================================

Запуск: дважды щёлкните VPN_Obhod_app.exe

Как пользоваться:
1. Вставьте ссылку ss:// в верхнее поле.
2. Нажмите «Подключить по ссылке».
3. Разрешите смену системного прокси при первом запуске (если спросит).
4. Весь трафик ПК пойдёт через прокси. Отключение — кнопка «Отключить».

Важно: папку v2ray не удаляйте — в ней ядро V2Ray (v2ray.exe). Без неё подключение по ссылке не заработает.

Распространение: копируйте всю папку VPN_Obhod_app целиком (exe + v2ray + все файлы).
